# 🧠 Multilingual Fake News Detection System

## 📌 Overview
This project is an AI-based system designed to detect fake news in **Tamil and Sinhala languages** using Natural Language Processing (NLP), machine learning models, and semantic similarity techniques.

It also includes a **source credibility scoring system** to evaluate the reliability of news sources.

---

## 🚀 Features
- Fake news detection for Tamil and Sinhala content
- NLP-based text preprocessing
- Semantic similarity matching
- Source credibility scoring system
- Ensemble-based prediction approach
- High-speed inference for real-time analysis

---

## 🧠 Tech Stack
Python | NLP | Scikit-learn | Pandas | NumPy | Machine Learning | Flask (optional API)

---

## 🏗️ System Architecture
- Data Collection & Preprocessing
- Feature Extraction (TF-IDF / embeddings)
- Classification Model
- Semantic Similarity Module
- Credibility Scoring Engine
- Final Ensemble Decision Layer

---

## 📊 Results
- Tamil model accuracy: ~83.9%
- Sinhala model accuracy: ~82.3%
- Source credibility precision: ~87.5%
